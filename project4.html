Ice Hockey Team Webpage Design | Rejects Sports

Tools Used: HTML, CSS, JavaScript (Shopify Platform), Photoshop, Adobe Illustrator

Project Overview: This project involves designing and coding a dynamic webpage for an ice hockey team on the Shopify platform. The goal is to create a visually engaging site that represents the team’s brand, highlights their achievements, and provides an intuitive experience for fans and visitors. The site includes custom product pages for team merchandise, a schedule of events, and team-related content. The design features a modern, responsive layout that enhances accessibility across all devices, creating a cohesive and interactive experience that reflects the team's spirit.

Project Objectives

  • Build a Team-Focused, Engaging Web Presence: Design a website that showcases team information, upcoming events, and player profiles, fostering fan engagement.
  • Optimize E-Commerce for Merchandise: Implement product pages and a seamless shopping experience for fans to purchase team apparel and accessories.
  • Ensure Cross-Platform Responsiveness: Create a responsive layout that works seamlessly on desktop and mobile devices.
  • Reflect Team Branding: Utilize visual elements and colors that align with the team’s identity, creating a consistent and memorable experience.

Project Outcomes

Website is continuously updated, current outcomes including:

  • Comprehensive and Interactive Webpage: A fully functional site featuring team information, merchandise, and event scheduling.
  • Enhanced Fan Engagement: An online platform that connects fans with the team through interactive features and regular updates.
  • Streamlined Shopping Experience: An intuitive Shopify-based e-commerce setup that provides a smooth and secure way to purchase team products.
